# Quasar.Agent/GridRenderSceneInspector.cs

**Module:** Quasar.Agent  **Kind:** class  **Tier:** 1

## Summary
Game-thread helper that builds viewer scene snapshots from live `MyCubeGrid` and asteroid `MyVoxelBase` instances. It maps primary and optional context grids into `Magnetar.Protocol` viewer DTOs, computes selected-grid-local context bounds plus matching relative and enclosing world AABBs, enumerates loaded intersecting grids deterministically, selects surrounding grids, blocks, and voxel bodies by projecting their bounds into the selected grid's relative view space and testing the same snapped floor-grid X/Z rectangle used by browser clipping, registers active scene mod asset roots and logical model/texture names with mod root hints, includes generated cube-part and runtime subpart metadata with subpart entity IDs, refreshes and exports captured emissive material states including light-specific render-property bulb updates, captures current-model orientation matrices plus build level, integrity, max integrity, accumulated damage, and damaged-block skeleton offsets, captures selected-grid conveyor/logistics topology without marking context-clipped but still connected endpoints as dangling, captures terminal-synced lighting and LCD metadata, assigns grid ownership on blocks/chunks/lights, samples voxel content/material ranges from the matching floor-footprint prism when requested, and emits separate voxel damage chunks with per-sample masks produced by comparing current storage against the voxel data provider's original terrain.
