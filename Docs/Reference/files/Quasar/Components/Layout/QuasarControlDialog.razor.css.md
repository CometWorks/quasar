# Quasar/Components/Layout/QuasarControlDialog.razor.css

**Module:** Quasar.Components  **Kind:** CSS  **Tier:** 3

## Summary
Scoped CSS for `QuasarControlDialog.razor`. Makes each power action a full-width, left-aligned button with stable height and wrapping title/body text for narrow mobile screens.

## Structure
Classes:
- `.quasar-control-action` — full-width action button, left-aligned, no uppercase transformation.
- `.quasar-control-action-title` — bold action label.
- `.quasar-control-action-body` — secondary explanatory line with normal wrapping.
