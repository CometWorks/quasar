# Quasar/Components/Pages/FolderPickerDialog.razor.css

**Module:** Quasar.Components  **Kind:** CSS  **Tier:** 3

## Summary
Minimal scoped stylesheet for `FolderPickerDialog.razor`. Contains a single rule that removes the default uppercase text transform from the breadcrumb navigation buttons so directory names render in their natural casing.

## Structure
- `.folder-picker-breadcrumb` — `text-transform: none` applied to `MudButton` elements in the breadcrumb row.
