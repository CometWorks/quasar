# Quasar/package.json

**Module:** Quasar.Host  **Kind:** npm manifest  **Tier:** 3

Exact npm dependency manifest for the standalone grid viewer browser runtime packages. The manifest is private and pins `three` `0.180.0` and `@zip.js/zip.js` `2.7.72`; `Quasar.csproj` restores it with `npm ci --ignore-scripts` and stages the full package directories into `wwwroot/vendor` for local static serving.
