# Quasar/package-lock.json

**Module:** Quasar.Host  **Kind:** npm lock file  **Tier:** 3

Npm lock file for the standalone grid viewer browser packages. It records the exact registry tarballs and integrity hashes for `three` `0.180.0` and `@zip.js/zip.js` `2.7.72`, preventing version drift or package-name substitution when the build stages `/vendor` static assets.
