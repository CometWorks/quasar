# Quasar/wwwroot/lib/uplot/uPlot.iife.min.js

**Module:** Quasar.Host  **Kind:** vendored JS  **Tier:** 3

## Summary
Vendored minified IIFE build of uPlot v1.6.32. It exposes the browser global `uPlot`, which Quasar uses for lightweight client-side analytics chart rendering.

## Structure
- Minified third-party JavaScript bundle.
- Defines chart construction, scales, axes, cursors, legends, path renderers, sync support, and date/number formatting helpers.
- Intended to be served unchanged from `wwwroot/lib/uplot/`.

## Dependencies
- Used by [`Quasar/wwwroot/quasar-charts.js`](../../quasar-charts.js.md)
- Styled by [`Quasar/wwwroot/lib/uplot/uPlot.min.css`](uPlot.min.css.md)

## Notes
This is a vendored asset. Prefer replacing it with a verified upstream uPlot release file instead of hand-editing the minified contents.
