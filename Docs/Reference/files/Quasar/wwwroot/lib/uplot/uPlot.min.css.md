# Quasar/wwwroot/lib/uplot/uPlot.min.css

**Module:** Quasar.Host  **Kind:** vendored CSS  **Tier:** 3

## Summary
Vendored minified stylesheet for uPlot. It provides the DOM layout and visual defaults required by uPlot charts rendered by Quasar's analytics chart interop.

## Structure
- Minified third-party CSS asset.
- Styles uPlot chart containers, axes, legends, cursors, selection overlays, and point/series elements.
- Intended to be served unchanged from `wwwroot/lib/uplot/`.

## Dependencies
- Applies to charts created by [`Quasar/wwwroot/lib/uplot/uPlot.iife.min.js`](uPlot.iife.min.js.md)
- Used indirectly by [`Quasar/wwwroot/quasar-charts.js`](../../quasar-charts.js.md)

## Notes
This is a vendored asset. Prefer replacing it with a verified upstream uPlot release file instead of hand-editing the minified contents.
